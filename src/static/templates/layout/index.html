Django-Angular-Base Starting Point for Dynamic Django Applications

The Django-Angular-Base template provides a solid start to a standard implementation of a Django powered API application using AngularJS to handle the frontend magic.

When developing Django applications nowadays, one of the biggest downsides is that making them dynamic can be a pain. With all the new JavaScript frameworks available, I needed a solid starting point to build new projects from this. This is why this project exists. With this structure, we can easily build Django applications with an AngularJS frontend. It provides out of the box frontend Authentication and gives a solid foundation for building additional apps and interfaces into it.

As well, it provides a server setup for easy deployment and management from a local filesystem. For a list of all the expected technology, check these out:

  • Django 1.7.3
  • Django Rest Framework 3.0
  • AngularJS 1.3
  • Bootstrap 3.3
  • CoffeeScript 1.8.0
  • SASS 3.4.10
  • UI-Router
  • UI-Bootstrap
  • Gulp
  • Fabric
  • NGINX
  • uWSGI
  • Supervisor
